Simple. Layout

This lay-out consists of HTML, SCSS, and vanilla JavaScript. It can be used anywhere, for any project. Customize the variables to automatically change all the colors necessary.

Features

  • Customizable colors
  • Semantic HTML
  • Fully Accessible (for screen readers and keyboard navigation)
  • Responsive HTML
  • Styling on all objects
  • Scrolling Effects
  • Google Material Icons
  • CSS Transitions and Animations
  • Dark Mode

Typography

Headings: Abril Display

Paragraphs, lists, links and figcaptions: Acumin Pro

Special: Flood
